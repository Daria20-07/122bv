#Прототип in-memory базы данных 
## Структура проекта 
- 'database.py' - логика хранения и поиска
- 'main.py' - интерфейс консоли
- 'README.md' - описание
## Функциональность
- Добавление записей (автоматическая генерация id)
- Вывод всех записей
- Фильтрация по одному/нескольким полям
- Обработка ошибок (неверный ввод,отосутствие записей)
## Запуск
```bash
python main.py
