#Лабораторная работа №3: In-memory БД с классами, тестами и сортировкой
##Структура проекта 
- 'models/' - классы Table, StudentTable, GradeTable
- 'storage/' - сохранение/загрузка JSON
- 'ui/' - консольный интерфейс (класс)
- 'tests/' - unit-тесты (покрытие>80%)
- 'main.py' - точка входа
##Функциональность
- Полный CRUD для двух таблиц (студенты и оценки)
- Валидация данных
- Защита от дубликатов
- Сортировка по любому полю (по возрастанию/убыванию)
- Сохранение и загрузка JSON
- Консольный интерфейс с переключением таблиц 
##Запуск
```bash
python main.py
